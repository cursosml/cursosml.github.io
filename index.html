Python para Ciencia de Datos

Aprende Python desde cero con un enfoque en aplicaciones de ciencia de datos. Este curso cubre manipulación de datos, visualización y técnicas básicas de aprendizaje automático.

Dominio del Machine Learning

Sumérgete en los algoritmos de aprendizaje automático, construcción de modelos y su implementación. Este curso está diseñado para aquellos que desean dominar las técnicas y herramientas de machine learning.

Ciencia de Datos para Ciencias Sociales

Dirigido a investigadores de ciencias sociales, este curso enseña análisis de datos, métodos estadísticos y técnicas de visualización específicas para datos de ciencias sociales.

Análisis de Series de Tiempo

Especialízate en la previsión y análisis de series de tiempo. Este curso cubre modelos ARIMA, suavización exponencial y técnicas avanzadas de redes neuronales para datos de series de tiempo.
